# 현장 테스트 추가 지시

## 테스트 건물 추가: 합동빌딩
현장 테스트를 위해 아래 건물 데이터를 DB에 우선 추가해주세요.

### 건물 정보
- 건물명: 합동빌딩
- 주소: 서울 강남구 역삼로 217
- 좌표: 위도 37.496789, 경도 127.040776
- Google Place ID: ChIJZ6Cyd9GnfDURnV2SOSj1wJI

### 요청사항
1. 이 건물을 buildings 테이블에 시드 데이터로 추가해주세요
2. 층별 정보(floors)는 더미 데이터로 만들어주세요 (일반적인 강남 오피스 건물 기준으로 적절히 생성)
3. 편의시설(facilities)과 건물통계(building_stats)도 더미로 추가
4. LIVE 피드도 2~3개 더미 추가
5. 앱 실행 시 이 건물이 가장 먼저 테스트되도록 해주세요

### 테스트 시나리오
직원이 합동빌딩 앞(역삼로 217)에서 테스트할 예정입니다.
1. 앱 실행 → 일반 모드 선택
2. 합동빌딩 방향으로 카메라를 비춤
3. 건물 핀 + 층별 오버레이가 표시되는지 확인
4. 핀 탭 → 하단 정보 카드가 나오는지 확인
5. 포인트 받기 버튼 동작 확인 (더미)
6. LIVE 섹션 표시 확인 (더미)

각 단계별로 성공/실패를 기록할 수 있는 체크리스트도 만들어주세요.
